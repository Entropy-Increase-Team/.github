## 关于我们

- **熵增项目组（Entropy-Increase-Team）** 是一个专注游戏生态周边工具的开发者集合：我们把「查数据、做推送、接多框架」做成可复用的一条链路，让更多玩家在 社交平台 机器人、桌面端、自动化等场景里用得更顺手。

- 项目多数以 **GNU AGPLv3** 开源；欢迎 Star、Issue 与 PR —— 你的支持是我们持续维护的动力。

> [!TIP]
> 名称里的「熵增」指向热力学里的不可逆过程：我们希望用清晰工程与开放协作，把杂乱需求收敛成可演进的产品 —— 让好用多一点，折腾少一点。

## 开放平台

绝大多数插件依赖统一的 **后端** 完成授权、密钥与接口调用：

### 开放平台统一入口

[落地页-开发中](https://open.shallow.ink)

### 各平台入口

| 分类 | 站点 | 说明 |
|:---|:---|:---|
| 三角洲行动 | [df.shallow.ink](https://df.shallow.ink) | 注册登录、创建 API Key、订阅与个人信息 |
| 明日方舟：终末地 | [end.shallow.ink](https://end.shallow.ink) | 官网授权与开发者 API 密钥 |
| Wegame | 开发中 | 用于wegame系列插件的调用 |

> [!TIP]
> 各仓库 README 内会写明必填配置项（如 `api_key` / `clientID`）；使用前请先阅读对应插件的安装与配置章节。
